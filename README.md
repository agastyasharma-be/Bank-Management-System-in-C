# 🏦 Bank Management System in C

A menu-driven Bank Management System developed in **C** that allows users to perform basic banking operations. The project demonstrates the use of **structures, arrays of structures, functions, file handling, loops, and input validation**.
---
## 🚀 Features
- Create a new bank account
- Deposit money
- Withdraw money
- Transfer money between accounts
- View account details
- Change account PIN
- Delete an account
- Save account data using binary file handling
- Load saved account data automatically on startup
---

## 🛠️ Technologies Used
- C Programming
- Structures

- Functions

- Binary Files

## 💾 File Handling

The application stores account data in a binary file (`accounts.dat`) so that all account information persists even after the program is closed.
